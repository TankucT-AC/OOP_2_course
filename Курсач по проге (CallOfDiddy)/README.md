# CallOfDiddy - 2D Horror Survival Game

## Описание
**CallOfDiddy** — 2D-игра в жанре хоррор-адвенчуры с элементами выживания. Игрок управляет камерой наблюдения в тёмной комнате с ограниченной видимостью. Цель — выбраться из помещения, собрав необходимые предметы и избегая угроз.

**Техническая реализация**: Проект написан на чистом C++ с использованием Windows API:
- `<windows.h>` — базовые функции Windows
- `<gdiplus.h>` — графический вывод
- `<mmsystem.h>` — аудио и системные звуки

## Геймплей
### Сюжет
Вы — камера наблюдения в тёмной комнате. Ваша задача — собрать 5 ключей и сбежать, пока не закончился заряд батареи.

### Механики
- **Камера**: ограниченный радиус видимости (освещение вокруг камеры)
- **Батарея**: постепенно разряжается, при достижении 0 — игра окончена
- **Предметы**:
  - 🔑 **Ключ** (5 штук) — нужны для открытия выхода
  - 🔋 **Аккумулятор** — восстанавливает заряд батареи
  - ✝️ **Крест** — при столкновении переключает на другую камеру
- **Враги**:
  - 🍼 **Детское масло** — преследует игрока
  - **Скримеры** — случайные пугающие события с изображением Пидиди

## Управление
- **W, A, S, D** или **Стрелки** — движение камеры
- **E** или **Пробел** — взаимодействие с объектами
- **ESC** — пауза/выход в меню
- **F** — фонарик (если реализовано)

## Системные требования
### Минимальные
- **ОС**: Windows 10/11 (x64)
- **Процессор**: 1.8 GHz Dual Core
- **Оперативная память**: 2 GB RAM
- **Место на диске**: 200 MB

### Рекомендуемые
- **ОС**: Windows 11 (x64)
- **Процессор**: 2.4 GHz Quad Core
- **Оперативная память**: 4 GB RAM
- **Место на диске**: 500 MB

## Установка и запуск

### Важно для Windows 11!
На **Windows 11** требуется запускать исполняемый файл **в режиме администратора**, так как старый терминал (консоль) был вырезан как устаревший компонент системы.

### Шаги установки:
1. Скачайте архив с игрой
2. Распакуйте в любую папку (желательно без кириллицы в пути)
3. **Правой кнопкой мыши** по `CallOfDiddy.exe`
4. Выберите **"Запуск от имени администратора"**

### Альтернативные варианты запуска:
- Создайте ярлык и в свойствах укажите "Запуск от имени администратора"
- Запустите через PowerShell с правами администратора:
```powershell
Start-Process "CallOfDiddy.exe" -Verb RunAs
